# @ticnat/nativescript-t3b-printer

Supported platforms:
#iOS 

```javascript
ns plugin add @ticnat/nativescript-t3b-printer
```

## Usage

```javascript
this.printer = new T3bPrinter();

this.printer.connectWifi("192.168.1.240");

this.printer.printTxt("some text");
```

for full example see demo.
